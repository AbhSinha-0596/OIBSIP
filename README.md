## 📂 Project Deep Dives

### 🛡️ Task 1: Simple Calculator App
A modern calculator app to help users in calculations such as addition, subtraction, multiplication, division with real-time feedback

* **Key Engineering Features:**
    * **Fluid Responsive Layouts:** Eradicated layout shifting via strict CSS Grid and Flexbox orchestration without relying on heavy UI frameworks.
    * **History Panel:** History panel to store recently executed calculations for review
* **Key Learnings:** Mastery of CSS lifecycle management, cross-browser compatibility debugging, and semantic HTML5 structuring.

### 🛡️ Task 2: Dynamic To-Do Web Application
An interactive client-side automation engine built entirely on native web APIs, empowering users with task control metrics and persistent session management.

* **Key Engineering Features:**
    * **Advanced DOM Reconciliation:** Handled high-frequency UI tracking via semantic Event Delegation on container parents, lowering browser memory profiles.
    * **Robust Management:** CRUD operations are performed along with management of real-time feedback and smooth interaction.
* **Key Learnings:** Improved understanding of JavaScript execution contexts, DOM node mutation methods, and clean object arrays handling.

### 🛡️ Task 3: Tribute Page
A webpage to pay honorary tribute to an inspiring role-model and a visionary - Shri Ratan Tata.
* **Key Engineering Features:**
    * **Sequential Divisions** Events are sequentially arranged to represent lifecycle and big-impacts made by him.
    * **Theme Attribution:** Webpage theme is made for the attribution of the his vision, group and achievements.
* **Key Learnings:** Managing subtle CSS components to display his life-events and inspiring values.

### 🛡️ Task 4: Login-Authentication
A web security page to allow users to access a protected pages only after login.
* **Key Engineering Features:**
    * **Global State Composition:** Deployed React Context API and `useContext` architectures to access and distribute session data for authentication.
    * **Component Modularity:** Designed highly reusable UI tokens packaged cleanly inside Tailwind CSS styling systems.
    * **Password Hashing:** Utilizing JWT based password hashing before storing user data within database (MongoDB) 
    * **OTP-based Verification:** OTP based email verifcation for first time login and for reseting passwords to improve security.
    * **Secure Logout:** Removing login-credentials from device securely when user clicks logout in webpage
* **Key Learnings:** Managing server-side functions, routes, controllers and cross-site operations and client-side routes and GET, POST requests for accessing and verifyinf user credentials in database.
