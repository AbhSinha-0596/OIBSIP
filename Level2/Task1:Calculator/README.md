## 🧮 Calculator App

A  responsive, and fully functional web-based calculator built using **HTML, CSS, and JavaScript**. This project was developed as part of my **Oasis Infobyte Web Development Internship** task.

---

## ✨ Features
* **Basic Arithmetic Operations:** Supports addition, subtraction, multiplication, and division.
* **Advanced Keys:** Includes standard calculator functions like pi (`π`), clear (`C`), and backspace/delete.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
* **History Panel:** History panel to store recent equations for review

---

## 🛠️ Tech Stack
* **HTML5:** Structure and semantic layout.
* **CSS3:** Styling, grid/flexbox layout, and responsive media queries.
* **JavaScript (ES6):** Core logic, event handling, and mathematical evaluations.
